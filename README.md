## Пробное приложение для изучения FastAPI

### Тербования

Python 3.9.0

### Установить зависимости 

```
python.exe -m venv venv --upgrade-deps
.\venv\Scripts\activate
pip install -r requirements.txt
```

### Запуск

python main.py

### Доступ к Swagger

http://127.0.0.1:8001/docs   (или другой указанный в коде приложения URL)
